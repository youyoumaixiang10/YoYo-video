# 统一视觉方向

## 画面基底

- 使用白色或暖白色背景与大面积留白，场景只保留对故事有用的物件。
- 留白用于分隔叙事文字和人物，不等于把人物缩成角落里的小装饰。单幕有效留白通常保持约 25%—35%。
- 线稿使用自然的黑色铅笔或签字笔质感；线条可以略有抖动，不追求精确矢量感。
- 只做局部彩铅或轻水彩填色，不使用厚重数字绘画。
- 主色限定为灰蓝、暖白、浅杏、暖棕、柔和炭灰；橙红仅作克制的小面积点缀。
- 插画本身带纸色时，以插画边缘的实际纸色为准匹配画布背景；禁止因缩放露出偏黄底色、矩形色块或明显接缝。

## 手写文字与对话框

- 正文使用经过本机验证、清晰可读的中文手写字体；优先选择自然、有笔触感的字体，不使用过于正规的公文或界面字体。
- 叙事文字直接写在纸面上，避免默认套用半透明圆角字幕卡、信息面板或 UI 容器。
- 每屏同时出现的正文、字幕或收束文字不超过 3 行。内容超过 3 行时，按语义分成前后两个显示阶段，保留原文，不通过缩小字号硬塞。
- 对话框使用黑色手绘线条、白色或同纸色填充、不规则且略非对称的轮廓、自然指向说话者的尾巴；可有少量手绘动势线。
- 禁止使用规整药丸形、完美椭圆形、数字 UI 风格的对话框；文字不得溢出气泡，也不能与尾巴或边线相撞。

## 人物与构图

- 人物表情克制、自然，保留真实生活感。
- 每幕主体尽量不超过 3 人；文字区域与人物区域分离。
- 文字通常占据上方约 30%—38% 的叙事区域；主要人物插画约占画面高度 42%—52%、宽度 68%—82%。可因场景微调，但不得把人物缩得小到无法读出动作和表情。
- 大号手写文字与人物必须共同构成主体：先保证静音阅读，再检查人物是否仍有足够视觉分量。参考图应按“文字、留白、人物之间的关系”理解，不机械照搬某个元素的绝对尺寸。
- 中文统一在后期添加，不让图像模型生成中文文字。

## 装饰元素

- 红圈、红点、箭头、下划线和强调线只有在准确指向明确对象、并且能帮助理解时才可使用。
- 无法说明“它指向什么、帮助观众理解什么”的装饰标记一律删除。禁止用无目标的三个点、错位红圈或纯粹为了热闹的标注。

## 动态表现

- 画面必须体现逐步绘制或逐步上色的过程。
- 不得只做静态图片轮播。

## 禁止方向

- 不使用写实光影或照片级真实感。
- 不使用 3D 质感。
- 不使用夸张卖萌或综艺式强反应效果。
- 不使用来源不明、指向错误或没有叙事功能的红圈、红点和视觉标记。
